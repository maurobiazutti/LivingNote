# LivingNote 📓

O **LivingNote** é um sistema web pessoal e privado desenvolvido para centralizar o controle de três pilares fundamentais da minha rotina: **Finanças**, **Estudos** e **Pensamentos Íntimos**. 

O objetivo é criar uma ferramenta sob medida, focando em performance, simplicidade visual e total controle sobre os meus dados.

---

## 🛠️ Stack Tecnológica

* **Backend:** Ruby on Rails (v8.0+)
* **Banco de Dados:** PostgreSQL
* **Frontend:** Tailwind CSS (via Importmaps/Propshaft)
* **Estilização & UI:** Clean, Dark Mode-first, focado em scannability.

---

## 🧭 Os Três Pilares

### 💰 1. Finanças (Finance)
* Controle de receitas, despesas e transferências.
* Categorização inteligente de gastos (Essenciais, Lazer, Estudos, Investimentos).
* Dashboard visual com progresso de metas de economia.

### 📚 2. Estudos (Academy)
* Cadernos de anotações organizados por tópicos de tecnologia e ciência da computação.
* Suporte a Markdown/Rich Text para escrita rápida de resumos.
* Registro de sessões de foco (Tempo dedicado por assunto).

### 🧠 3. Diário & Pensamentos (Reflections)
* Registro diário livre para descarga mental.
* Monitoramento de humor (*mood tracking*).
* Segurança reforçada para garantir privacidade total dos pensamentos íntimos.
